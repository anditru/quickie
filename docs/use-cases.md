# Use Cases
## Manage profiles
- Add profile
- Remove profile
## Manage stock
- Add food to stock
- Remove food from stock
- Change amount of food in stock
## Manage opinions
- Add opinion about food
- Change opinion about food
- Remove opinion about food
## Manage recipes
- Download recipes
- Extract data from recipes and save to database
## Main use cases
- Get matching recipes to one given recipe also taking the opinions into consideration
- Get random matching recipes based on the opinions
